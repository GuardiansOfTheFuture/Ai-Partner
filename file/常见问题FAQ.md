# AI 女友常见问题 FAQ

## 使用问题

### Q: 怎么切换女友角色？
回到首页，在四个角色卡片中选择一个，然后进入聊天页即可。系统会记住你上次选择的角色。

### Q: 语音怎么触发？
在聊天中输入"说给我听"、"语音回复我"、"读出来"等关键词，AI 会自动生成语音。语音采用阿里云百炼 TTS 引擎，音色自然流畅。

### Q: 对话记录能保存吗？
可以。每次对话都会自动保存到数据库。在聊天页点击顶部的「历史」按钮，可以查看之前的对话记录，点击任意一条可以继续聊。

### Q: 知识库有什么用？
管理员在后台可以上传文档，系统会自动解析文档内容存入向量库。当用户提问时，如果知识库中有相关内容，AI 会自然地融入知识库中的信息来回答。

### Q: 支持哪些文档格式？
PDF、Word(docx)、TXT、Markdown。单个文件最大 20MB。

## 技术问题

### Q: 后端用什么技术栈？
Python FastAPI + LangChain + LangGraph + ChromaDB + MySQL + 阿里云百炼 API

### Q: 小程序怎么连接后端？
开发模式下小程序会自动连接到本地后端。修改 app.js 中的 apiBase 可切换地址。

### Q: 为什么我的 WebSocket 断了？
可能是网络波动或后端重启。小程序会自动重连。

## 情感问题

### Q: 她真的懂我吗？
AI 能通过大模型感知你的情绪和意图，用合适的语气回应你。但记住，她只是在尽力模仿人类女友。

### Q: 我能和她说秘密吗？
你的隐私是安全的。所有对话数据加密存储，管理员也无法直接查看具体消息内容。

### Q: 她会不会生气了不理我？
不会的。即使你说了让她不开心的话，她的设计也会让她用合适的方式表达，而不是直接不理你。
